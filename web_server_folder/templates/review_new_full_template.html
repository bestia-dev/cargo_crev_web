Prerequisites to write new crev reviews for a crate version:
- on your developer machine install $ cargo install cargo-crev,
- on GitHub or GitLab fork the repo https://github.com/crev-dev/crev-proofs 
- clone it to your local machine
- create your personal crev ID $ cargo crev id new
Everything is encrypted, so be careful with the secrets, keys, passphrase,... There's no way to recover them.
Read the Get Start Guide to know better. 

The new crev review must be written with the command $ cargo crev crate review {crate_name} 
It opens the VIM editor. 
This is the important part: When saved, the review is encrypted automatically ! 
But writing in VIM is not easy for everybody. 
By the way type :q or :q! to exit or quit VIM. 
There is also the possibility to write in your favorite editor, that is much better, but not the best user experience. 
Here on this webpage are easy templates to write a review in the browser. 
The browser cannot finish the review with encryption. 
So you have to copy the resulting text to VIM: 
- first in the browser copy the resulting text, 
- run the command $ cargo crev id new. VIM will open.
- delete the first cc 10 lines with repeating the VIM shortcut dd for delete line 
- then in VIM type i to go into VIM Insert Mode 
- then mouse right click to paste 
- type : to go into VIM command mode            
- then type :wq to write and quit 
- now the encryption will kick in 
- Done. Easy.
